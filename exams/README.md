# Exams folder
Exams folder will keep everything related with the entrance informatics exams at Romanian Universities like UBB and TUIASI.

## Who is this folder for?
This repository is useful if you are:

- Currently preparing for entrance informatics exams at Romanian Universities.
- Solving problems in **pbinfo**
- Looking for **clean, fast C++ implementations** with explanations.

This repository is **not** intended for:

- Absolute beginners with no programming background.
- Copy-paste usage without understanding the solutions.
