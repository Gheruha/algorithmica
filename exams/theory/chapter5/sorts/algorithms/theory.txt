Inneficient sorting algorithms O(n^2): 

* Bubble sort:
  - We travel the vector, if there are 2 neighbor elements that are not in the correct order, we exchange them.
  - After one travel, not all the elements will be sorted, however we can do that as many times needed till there is nothing to exchange. 

* Select sort:
  - We take array[0].
  - We search travel the vector and take the minimum. 
  - We swap array[0] with the minimum.
  - Same steps but with array[1,2,...n - 1].

* Insertion sort:
  - let's say a vector X[] with n elements.
  - if the sequence with indices 0, 1, ... i-1 is ordered, then we can insert the element array[i] in this sequence so that the sequence with 
    indices 0, 1, ..., i-1, i is ordered.
  - we take each element array[i] in turn and insert it into the sequence to its left.
  - at the end the whole array will be ordered.


* Collating is what we use in Merge Sort:
  - We take 2 sorted arrays with n and m elements.
  - We travel them at the same time.
  - We select the value smaller from the two arrays.
  - We add the value in a third array.
  - We continue the travel in the array we took the element from.
  - At the end the travel of one of them will be done.
  - We take what's left and added to the third array.
  - Third array's size is m + n.


* Counting Sort:
	-	We create another array occurrences_array of size k + 1, where k is the maximum value in the original array (or max - min + 1 in the general case).
	-	We count the occurrences of each value from the original array and store them in occurrences_array (the index represents the value).
	-	For each value v, occurrences_array[v] stores how many times v appears in the original array.
	-	Then, we transform occurrences_array into a prefix sum array by adding each element with the previous one; after this step, occurrences_array[v] represents the number of elements less than or equal to v, which corresponds to the final position of v in the sorted array.
	-	We create a new array of size n to store the sorted result.
	-	We traverse the original array (preferably from right to left for stability), and for each element x, we place it in the position occurrences_array[x] - 1 in the new array, then decrement occurrences_array[x].
